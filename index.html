Que és

nubarchiva Alfresco es un AMP (Alfresco Module Package) para sistema de gestión documental Alfresco que permite adaptar el módulo de Record Management para la gestión archivística de archivos según la normativa española.


Descripción

Alfresco dispone de las capacidades de gestión documental que permiten a una entidad que lo implante disponer de su documentación organizada y fácilmente accesible. La comunidad de Alfresco ha desarrollado un módulo denominado Alfresco Record Management (RM) que permite la gestión archivística. Sin embargo este módulo se ha desarrollado cumpliendo con la normativa del departamento de defensa norteamericano DoD 5015.22 para gestión de archivos. Así pues lo hace insuficiente para su aplicación a la gestión de archivos en España.

Este módulo adapta Alfresco RM para adaptarlo a las necesidades de un archivo español incorporando las siguientes características:

  • Traducción completa al castellano
  • Metadatos descriptivos en formato multinivel ISAD
  • Metadatos de control para la gestión de prestamos
  • Materias
  • Incorporación de un cuadro de clasificación orientado a una administración pública
  • Manual de usuario para archiveros
